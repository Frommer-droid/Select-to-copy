# Select-to-Copy

Утилита для Windows, которая автоматически копирует выделенный текст после выделения мышью.
Приложение работает в системном трее и не требует открытия основного окна.

## Версия
`v1.12.0`

## Что делает приложение
- Отслеживает выделение текста левой кнопкой мыши.
- После завершения выделения автоматически копирует текст в буфер обмена.
- Показывает уведомление `Скопировано!` рядом с курсором.
- Позволяет поставить копирование на паузу через меню в трее.
- Поддерживает исключения по процессам: в выбранных приложениях автокопирование отключается автоматически.

## Поведение с клавишами
- Обычный режим: выделили текст мышью -> текст автоматически скопирован.
- Режим `Left Ctrl`: если выделять текст с зажатым **левым Ctrl**, текст **выделяется**, но **не копируется** (режим «только выделение»).

## Иконка в трее
- Активный режим: основная иконка `logo.ico`.
- Пауза: используется отдельная иконка `assets/icons/logo-pause.ico`.
- Левый клик по иконке открывает окно исключений.
- Пункт меню паузы работает как переключатель:
  - `Пауза` -> выключает автокопирование;
  - `Возобновить` -> снова включает автокопирование.

## Исключения по процессам
- Открыть окно исключений можно левым кликом по иконке в трее или через пункт `Исключения...`.
- В списке процессов поддержан выбор как в Проводнике Windows (`Ctrl`/`Shift`).
- Доступны `Импорт`/`Экспорт` списка исключений в JSON.
- Кнопка `Захватить процесс...` работает с полупрозрачным overlay:
  - после подтверждения есть 3 секунды, чтобы активировать нужное окно;
  - затем появляется overlay, клик по окну добавляет его процесс в исключения.
- Кнопка `Сохранить` применяет изменения без закрытия окна исключений.
- Внизу окна исключений есть `Status Bar` для коротких сообщений о действиях вместо всплывающих мини-окон.
- Когда активен процесс из исключений, приложение автоматически переходит в авто-паузу и автоматически выходит из нее при смене окна.

## Требования
- Windows 10/11
- Python 3.12+
- Установленные зависимости:
  - `PySide6`
  - `pynput`
  - `pyperclip`
  - `pywinauto`
  - `pywin32`

## Установка зависимостей
```bash
pip install PySide6 pynput pyperclip pywinauto pywin32
```

## Запуск из исходников
```bash
python Select-to-Copy.pyw
```

## Сборка `.exe`
- Основной способ: `Build_Tools/SpecCompiler.pyw`
- Используемый spec-файл: `Build_Tools/Select-to-Copy.spec`

Прямой запуск сборки через PyInstaller:
```bash
python -m PyInstaller Build_Tools/Select-to-Copy.spec
```

## Структура проекта (кратко)
- `Select-to-Copy.pyw` — точка входа.
- `app/main.py` — запуск `QApplication` и контроллера.
- `app/core/` — оркестрация приложения.
- `app/ui/` — UI и трей-меню.
- `app/services/` — логика копирования, обработка мыши, системные сервисы.
- `app/config/` — константы и тема.
- `assets/icons/logo-pause.ico` — иконка режима паузы.

## Логи
- Файл лога: `select_to_copy.log` в корне проекта.
- Лог очищается при старте новой сессии.

## Настройки
- Файл `settings.json` создается в корне приложения.
- В нем сохраняются параметры окна исключений и список процессов-исключений.

## Релизный процесс
- Версия хранится в `VERSION` и синхронизируется с `app/version.py`.
- История изменений ведется в `RELEASE_NOTES.md`.
