# История изменений

## [v1.12.1] - 2026-03-04
### 🐛 Исправления ошибок
- Исправлена логика окна исключений: кнопка `Сохранить` применяет настройки без закрытия окна, управление происходит через крестик окна.
- Устранена путаница с пунктом паузы в трее: текст переключателя теперь `Пауза/Возобновить` в соответствии с README.

### 🛠 Улучшения
- Обновлен UI окна исключений: удалена кнопка `Отмена`, выровнены размеры `Захватить процесс...` и `Сохранить`.
- Вместо всплывающих мини-окон добавлена нижняя строка состояния (`Status Bar`) для сообщений о действиях (`Сохранено`, импорт/экспорт, захват процесса).
- Добавлены стили темной темы для `QStatusBar` в окне исключений.

## [v1.12.0] - 2026-03-04
### ✨ Новые возможности
- Добавлено окно управления исключениями по процессам с темной темой (`Nightfall`) и сохранением состояния в `settings.json`.
- Реализована авто-пауза автокопирования при активных окнах процессов из списка исключений.
- Добавлен захват процесса через полупрозрачный overlay с задержкой 3 секунды перед стартом.
- Добавлено открытие окна исключений левым кликом по иконке в системном трее.
- Добавлены импорт и экспорт списка исключений в JSON.

### 🐛 Исправления ошибок
- Исправлен сценарий захвата процесса: добавлен надежный fallback определения процесса (окно под курсором -> foreground).
- Устранены проблемы отображения и взаимодействия с overlay в модальном диалоге исключений.

### 🛠 Улучшения
- Обновлен UI окна исключений: равномерный ряд из 4 кнопок, русский язык интерфейса и корректные цвета кнопок без белых окантовок.
- Добавлены unit-тесты для сервисов исключений, паузы и `settings.json`.
- Обновлен `README.md` под функциональность `v1.12.0`.
- Добавлены `LICENSE` (MIT) и обязательные секции `Автор`/`Лицензия` в конце `README.md`.

## [v1.11.0] - 2026-03-03
### ✨ Новые возможности
- Приложение переведено на `PySide6` с новой модульной архитектурой (`app/core`, `app/ui`, `app/services`, `app/config`).
- Контекстное меню трея обновлено: пункт паузы работает как переключатель без галочки.
- Для режима паузы добавлена отдельная иконка `assets/icons/logo-pause.ico`.
- В сборку добавлено включение каталога `assets` в `_internal`.

### 🐛 Исправления ошибок
- Исправлена регрессия автокопирования после выделения текста.
- Исправлен fallback `Ctrl+C` через `SendInput` со сканкодами на Windows.
- Устранен конфликт COM-инициализации (`OleInitialize`) при запуске Qt + UIA.

### 🛠 Улучшения
- Обновлен `README.md` под текущую версию и поведение трея.
- Обновлен `.spec` для новой архитектуры и ресурсов.
